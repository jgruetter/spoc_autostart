# SPOC autostart
## Ein BAT Script, welches die nögigen Tools für den Einsatz im SPOC öffnet

**Funktion:**
Die Datei öffnet die nögigen Tools für den Einsatz im SPOC. Es öffnet die nötigen Websiten, danach die Apps eins nach dem anderen, somit Admin-Logins eingegeben werden können.
Die Websiten werden mit dem Standard-Browser geöffnet. Ausnahme: Agent-Portal wird immer in Chrome geöffnet

**Welche Apps/Websiten werden geöffnet?**
Websiten:
- SRG SRR Workplace
- SRG SSR GetHelp
- Tim - Ticketing Management
- tpc-helpdesk Sharepoint
- Agent-Portal (Chrome)

Apps:
- SCCM Remote Control
- Remote Desktop Connection
- TeamViewer
